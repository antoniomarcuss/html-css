Trabalhando com listas

Listas Ordenas

Para fazer uma lista ordenada usamos a tag ol e dentro da tag ol usamos a tag li
para colocar cada palavra em uma lista.

Usamos também a tag type dentro da tag do ol para selecionar qual tipo de numeração você quer que apareça e a tag start serve para você selecionar qual numeração quer que começe, EXEMPLO

Ao acorda eu:

  1. tomo café
  2. escovo os dentes
  3. vou trabalhar
  4. não tomo banho de manhã por que a água é gelada

Listas não Ordendas

São aquelas sequências que não faz sentido,não preciso enumerar.

Para usar usamos a tag ul com o mesmo pricipio de raciocínio do ol EXEMPLO.

  • pão
  • leite
  • alface
  • banana
  • desodorante
